# AI-Clippings

Reusable Codex skill assets for AI clipping workflows.

Current skill:
- `obsidian-web-clipper-ingest/`: fully automates clipping article links into Obsidian through the Chrome Obsidian Web Clipper extension, including full-page scrolling, parallel summary waiting, and vault-level import verification.
- `wechat-filehelper-link-ingest/`: extracts non-video links from the Windows WeChat desktop app File Transfer Assistant, keeps a local link index, and supports time-range queries for clipped article URLs.
